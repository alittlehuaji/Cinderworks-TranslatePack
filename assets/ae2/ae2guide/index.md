# 什么是应用能源2？

应用能源2添加了各种组件和机制，以提供物流和存储解决方案。你可以用一个紧凑的ME网络取代堆满箱子的巨大房间，但这仅仅是开始。
应用能源旨在与模组包中的其他模组配合使用并实现自动化。你可以设置系统，只需一次点击，就能合成复杂合成链中的所有前置物品（和最终产物），或者保持一定数量的物品库存，在需要时自动合成更多物品，或者简单地在基地中传输物品。
