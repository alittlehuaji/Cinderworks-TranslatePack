# 福鲁伊克斯水晶

<ItemImage id="fluix_crystal" scale="4" />

*"福鲁伊克斯水晶拥有吸收并转化能量的独特能力，是所有物质能量技术的基础。"*

AE2方块、[设备](../ae2-mechanics/devices.md)和物品的主要合成材料之一。通过将下界石英、红石和<ItemLink id="charged_certus_quartz_crystal" />扔入水中来制作。

可以使用<ItemLink id="formation_plane" />和<ItemLink id="annihilation_plane" />来实现[自动化生产](../example-setups/throw-in-water-automation.md)。
