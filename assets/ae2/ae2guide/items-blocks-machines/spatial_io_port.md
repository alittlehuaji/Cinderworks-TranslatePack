# 空间IO端口

<BlockImage id="spatial_io_port" p:powered="true" scale="8" />

空间IO端口用于[空间IO](../ae2-mechanics/spatial-io.md)操作，它可以放置[空间元件](spatial_cells.md)并控制空间IO操作。

如果你想实现自动化，可以通过任何物品物流设备（如漏斗或AE2总线）来插入和取出元件。
