# 迷你TNT

<BlockImage id="tiny_tnt" scale="8" />

用于小规模爆炸的小型TNT。可用于制作<ItemLink id="quantum_entangled_singularity" />对。

你可以在配置中禁用方块破坏效果，这样就能在不影响服务器安全的情况下制作奇点，同时也可以禁用TNT和苦力怕。
