# 无线终端

<Row>
  <ItemImage id="wireless_terminal" scale="4" />

  <ItemImage id="wireless_crafting_terminal" scale="4" />
</Row>

无线终端是有线[终端](terminals.md)的便携版本。它们的界面与有线终端完全相同，但用[升级卡](upgrade_cards.md)槽位取代了<ItemLink id="view_cell" />槽位。

要将它们与网络配对，请将终端插入连接到该网络的<ItemLink id="wireless_access_point" />右上角的槽位中。（即显示无线终端图标并带有向下箭头的槽位）

它们必须在<ItemLink id="wireless_access_point" />的范围内才能工作。

它们的能源可以在<ItemLink id="charger" />中充能。

# 无线终端

<ItemImage id="wireless_terminal" scale="4" />

基础终端，现在可以随身携带了！在<ItemLink id="wireless_access_point" />范围内，查看并访问[网络存储](../ae2-mechanics/import-export-storage.md)中的物品，并从[自动合成](../ae2-mechanics/autocrafting.md)系统中请求制作物品。

参见[终端](terminals.md)

无线终端支持以下[升级卡](upgrade_cards.md)：

# 无线合成终端

<ItemImage id="wireless_crafting_terminal" scale="4" />

无线合成终端与普通无线终端类似，拥有相同的设置和功能区域，但增加了一个合成格，可以从[网络存储](../ae2-mechanics/import-export-storage.md)中自动补充材料。Shift+点击输出时请小心！

## 界面

参见[终端](terminals.md)

## 升级

无线合成终端支持以下[升级卡](upgrade_cards.md)：

*   <ItemLink id="energy_card" /> 用于增加电池容量
