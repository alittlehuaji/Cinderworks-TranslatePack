# 染色器

<ItemImage id="color_applicator" scale="4" />

染色器用于给可上色的方块涂色，如[线缆](cables.md)、羊毛、陶瓦、玻璃和混凝土。它使用[染色球](paintballs.md)或染料，雪球可以用来清除线缆上的颜色和方块上的染色球痕迹。

它的能源可以在<ItemLink id="charger" />中充能。

染色器类似于[存储元件](storage_cells.md)，其染色存储最便捷的填充方式是将染色器放入<ItemLink id="chest" />的存储元件槽位中。

右键点击目标方块以应用所选颜色。要切换颜色，按住Shift时使用滚轮，或在没有目标时右键点击。

## 升级

染色器支持以下[升级](upgrade_cards.md)，通过<ItemLink id="cell_workbench" />安装：

*   <ItemLink id="equal_distribution_card" />为每种类型分配相同的元件字节空间，使单一类型无法占满整个元件
*   <ItemLink id="void_card" />在元件已满时（或在使用均分卡时该特定类型的分配空间已满）销毁放入的物品。请注意进行分区设置！
*   <ItemLink id="energy_card" />以增加电池容量
