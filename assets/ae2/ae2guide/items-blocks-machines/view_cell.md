# 查看元件

<ItemImage id="view_cell" scale="2" />

查看元件用于筛选[终端](terminals.md)中显示的物品。你可以在<ItemLink id="cell_workbench" />中对其进行分区设置。

例如，假设你希望终端只显示一部分石材类建筑材料。将查看元件分区设定为这些材料，然后将其放入终端中，就只会显示这些物品了。

查看元件的效果是叠加的。如果你有一个用于橡木木板的查看元件和一个用于圆石的查看元件，将两者同时放入终端，就会同时显示木板和圆石。
