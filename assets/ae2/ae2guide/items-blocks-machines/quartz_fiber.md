# 石英纤维

<GameScene zoom="8" background="transparent">
<ImportStructure src="../assets/assemblies/quartz_fiber.snbt" />
<IsometricCamera yaw="195" pitch="30" />
</GameScene>

石英纤维用于在[网络](../ae2-mechanics/me-network-connections.md)之间共享能源，同时保持它们相互独立。
这允许你为[子网络](../ae2-mechanics/subnetworks.md)供电，
而无需到处放置能源接收器和能源线缆。它也可以用来防止线缆相互连接，
不过使用不同颜色的线缆或使用<ItemLink id="cable_anchor" />会更便宜且更有效。

它们是[线缆子部件](../ae2-mechanics/cable-subparts.md)。
