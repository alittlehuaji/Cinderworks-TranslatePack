# 物质球

<ItemImage id="matter_ball" scale="4" />

一个由普通物质构成的球体，可用作<ItemLink id="matter_cannon" />的弹药，或用于生产[彩弹球](paintballs.md)。

在<ItemLink id="condenser" />的物质球模式下，使用256个物品或桶装液体制造。
