# 熵变机械臂

<ItemImage id="entropy_manipulator" scale="4" />

熵变机械臂允许你通过右键点击和Shift+右键点击分别加热和冷却物体。它能做的事情不多，主要包括：蒸发或冻结水、将岩浆凝固成黑曜石、在世界中将原木烧制成木炭，以及将圆石烧制成石头。

如果目标方块没有特定的交互效果，它将充当打火石使用。

它的能源可以在<ItemLink id="charger" />中充能。
